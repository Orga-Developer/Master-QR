# Google AdMob Compliance Note

This application includes AdMob as a monetization layer and therefore must maintain a fully coherent advertising disclosure posture. The build should disclose the presence of ads, the use of advertising identifiers, and any personalization or consent gating relevant to the user’s region. AdMob configuration must not be silently enabled in a way that conflicts with local privacy notices or platform data-safety declarations. The application should avoid collecting unnecessary signals and should ensure that consent state is respected before any personalized advertising configuration becomes active. If an ad request is restricted, non-personalized or context-only serving should be used as the safer fallback. Revenue generation must never override transparency, age-related protections, or the user’s ability to manage consent options.
